# Shiksha Saathi Angular Frontend

## Run

```bash
npm install
npm run dev
```

App URL:
- `http://localhost:4200/login`

## Notes

- API calls use `/api/*` and are proxied to `http://localhost:4000` via `proxy.conf.json`.
- Auth/session keys match the legacy SPA (`shiksha_token`, `shiksha_user`).
- Student test page supports browser SpeechRecognition and offline queue sync.
